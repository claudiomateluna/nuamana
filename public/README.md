# PWA Icons

To make your Next.js app a PWA, you need to create the following icon files in this directory:

1. `icon-192x192.png` - 192x192 pixels icon
2. `icon-512x512.png` - 512x512 pixels icon  
3. `icon-1024x1024.png` - 1024x1024 pixels icon

You can create these icons using any image editing software or online tools like:
- Figma
- Canva
- Online PNG icon generators

For development purposes, you can also use placeholder images from services like:
- https://placehold.co/192x192
- https://placehold.co/512x512
- https://placehold.co/1024x1024
